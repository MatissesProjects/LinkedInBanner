# Track 1 Specification: Environment & Authentication Setup

## Goals
- Establish a Python 3.10+ development environment.
- Install and initialize Playwright for browser automation.
- Securely handle the `li_at` session cookie for passwordless authentication.
- Verify successful login by capturing a screenshot of the LinkedIn home page.

## Deliverables
- `requirements.txt` with necessary dependencies.
- A test script `src/auth_check.py` to verify authentication.
- Successful screenshot confirming login status.
