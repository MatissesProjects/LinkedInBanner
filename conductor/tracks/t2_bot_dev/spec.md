# Track 2 Specification: LinkedIn Bot Development

## Goals
- Identify the UI elements on the LinkedIn profile page required for banner updates.
- Develop a robust Playwright script to automate the banner upload process.
- Ensure the script handles potential UI changes or delays gracefully.

## Deliverables
- `src/linkedin_bot.py` containing the `update_banner` function.
- Verified banner update on the live LinkedIn profile.
